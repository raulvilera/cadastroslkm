Portal Lydia Kitz Moreira — Guia de Implantação

📋 O que foi criado

Um Google Apps Script dedicado exclusivamente para receber e registrar as avaliações dos usuários (gestores e professores) na aba AVALIAÇÃO DOS USUÁRIOS da planilha.

✅ Arquivo criado: google-apps-script-avaliacoes.js — na pasta do projeto

1 Abrir o Google Apps Script

Acesse: https://script.google.com

Clique em + Novo projeto para criar um projeto em branco.

💡 Crie este projeto separado do script principal das ocorrências. Isso deixa cada funcionalidade independente e mais fácil de manter.
2 Colar o código

No editor que abrir, apague todo o conteúdo existente e cole o conteúdo completo do arquivo:

📄 Arquivo a copiar: google-apps-script-avaliacoes.js (está na pasta do projeto)

Após colar, renomeie o projeto para algo como: LKM — Avaliações da Plataforma

3 Implantar como Web App

No menu superior, clique em: Implantar → Nova implantação

Configure assim:

Tipo de implantação : App da Web
Executar como       : Eu mesmo (sua conta Google)
Quem tem acesso     : Qualquer pessoa

Clique em Implantar e autorize as permissões solicitadas.

⚠️ Na primeira vez, o Google pedirá autorização. Clique em "Avançado" → "Acessar LKM — Avaliações" → "Permitir".
4 Copiar a URL gerada

Após a implantação, uma URL será exibida no formato:

https://script.google.com/macros/s/XXXXXXXXXXXXXXX/exec

Copie essa URL.

5 Colar no código da plataforma

Abra o arquivo:

services/sheetsService.ts

Localize a linha:

const RATING_SCRIPT_URL = 'COLE_AQUI_A_URL_DO_SCRIPT_DE_AVALIACOES';

Substitua pelo valor copiado no passo 4:

const RATING_SCRIPT_URL = 'https://script.google.com/macros/s/XXXXXXX/exec';
✅ Pronto! A plataforma passará a enviar avaliações diretamente para este script dedicado.
6 Publicar no GitHub e Vercel

Após alterar o sheetsService.ts, o código precisa ser publicado.

Execute no terminal (dentro da pasta do projeto):

git add services/sheetsService.ts
git commit -m "feat: URL do script dedicado de avaliacoes configurada"
git push

A Vercel fará o redeploy automático em 1–2 minutos.
